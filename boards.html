Program Learning Objectives

The following objectives reflect the core technical competencies developed through coursework and applied projects within my degree program.

  • Objective 1 – Python Programming (CSC203): Design, implement, and debug Python-based applications using structured and object-oriented programming techniques.
  • Objective 2 – Artificial Intelligence Foundations (CSC215): Apply foundational AI concepts including intelligent agents, search, and decision-making systems to real-world problems.
  • Objective 3 – Machine Learning (DAT301): Develop and evaluate machine learning models using supervised and unsupervised techniques to extract insights from data.
  • Objective 4 – Network Security (NET320): Analyze network vulnerabilities and implement security controls to protect systems, data, and communications.
  • Objective 5 – Advanced Programming (CSC310): Apply advanced programming principles, design patterns, and software architecture concepts to build scalable and maintainable systems.
  • Objective 6 – Database Concepts (DBA200): Design, implement, and manage relational databases using normalization, queries, and data integrity best practices.
